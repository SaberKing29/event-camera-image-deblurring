## 简介

事件相机提供了一种捕捉视觉信息的新方法，尤其在高速运动和低光照等挑战性条件下表现优异。本项目旨在利用事件相机的优势来改进图像去模糊的效果。**更多的背景信息和详细说明将会在未来更新。**
